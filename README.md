# Palestra Eagle's Eye: visão computacional aplicada à drones

> **Semana da Tecnologia 2026.1 - UNICAP**

Repositório criado com o intuito de armazenar os materiais utilizados na palestra de Visão Computacional Aplicada a Drones na Semana da Tecnologia de 2026.1 — evento realizado na UNICAP.

## Sobre o Projeto
O projeto **Eagle's Eye** demonstra a integração entre hardware de VANTs (Veículos Aéreos Não Tripulados) e softwares de Inteligência Artificial. O objetivo da apresentação é explicar de forma simples e acessível a estrutura física dos drones e mostrar, na prática, como o **Processamento Embarcado** e a **Visão Computacional** permitem que a máquina identifique obstáculos e tome decisões autônomas em milissegundos.

## O que você vai encontrar aqui

*   **Slides da Palestra:** Todo o material visual utilizado durante a apresentação (Estrutura, Hardware, Processamento de Imagem e Machine Learning).
*   **Apostila de Apoio:** Material em PDF detalhando os conceitos teóricos para quem quer aprofundar os estudos.
*   **Scripts Python (OpenCV):** Os códigos utilizados na demonstração ao vivo do processamento de imagem, incluindo:
    *   Leitura de imagem colorida.
    *   Conversão para Escala de Cinza.
    *   Aplicação de Suavização (Gaussian Blur).
    *   Detecção de Bordas (Filtro Canny).
    *   *(Bônus)* Rastreamento de movimento com Subtração de Fundo (MOG2).

## Como testar os códigos na sua máquina

Para rodar os exemplos de Visão Computacional de forma local, você precisará do Python e da biblioteca OpenCV instalados.

1. Clone este repositório:
   ```bash
   git clone [https://github.com/Jhon-Victor-Ramos/palestra-eagles-eye.git](https://github.com/Jhon-Victor-Ramos/palestra-eagles-eye.git)
   ```
2. Instale as dependências:
   ```bash
   pip install opencv-python
   ```
3. Execute os scripts dentro da pasta `/codigos`.
